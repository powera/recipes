# Recipes Repository

This repository contains recipes and food reference data stored as Markdown (`.md`) files.

## Structure

### `recipes/`

Recipes organized by category:

- `recipes/bread/` — Bread recipes
- `recipes/curries/` — Curry recipes
- `recipes/salads/` — Salad recipes
- `recipes/soups/` — Soup recipes
- `recipes/stews/` — Stew recipes

Each recipe is a standalone `.md` file within its category directory.

### `foods/`

Food reference files organized by type. Each file describes a specific product with a short description and price data for a standard size.

- `foods/produce/` — Fresh produce (vegetables, tofu, etc.)
- `foods/dairy/` — Dairy products (eggs, sour cream, etc.)
- `foods/cheese/` — Cheese (cheddar, mozzarella, etc.)
- `foods/canned/` — Canned and shelf-stable goods (beans, coconut milk, broth, etc.)
- `foods/snacks/` — Snack foods (chips, popcorn, etc.)
